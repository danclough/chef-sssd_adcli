sssd_adcli Cookbook CHANGELOG
========================
This file is used to list changes made in each version of the sssd_adcli cookbook.

v1.0.2 (2020-08-07)
------------------
- Added new functionality to SSSD config template including default/fallback shell, home directory and more.

v1.0.1 (2018-08-03)
------------------
- Added support for managing PAM mkhomedir functionality for AD users.

v1.0.0 (2020-08-01)
------------------
- Forked from the [Biola University sssd_ad cookbook](https://github.com/biola/chef-sssd_ad).
- Retooled to use adcli instead of Samba.
